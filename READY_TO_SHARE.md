2. Click "Get Started"
3. Enter name, email, password
4. ✅ Get 50 welcome coins instantly

### **Step 2: Daily Check-in**
1. Go to Dashboard
2. Click "Check In" button
3. ✅ Earn coins + build streak
4. 🎉 See confetti + sound effect

### **Step 3: Play Games**
1. Click "Earn Coins" in navbar
2. Try all 3 games:
   - **Quiz**: Answer 5 questions
   - **Spin Wheel**: Daily spin
   - **Scratch Card**: 3 attempts
3. ✅ Earn coins for each game

### **Step 4: Complete Tasks**
1. Go to "Tasks" tab
2. Click "Complete" on any task
3. ✅ Earn coins instantly

### **Step 5: Browse Shop**

3. View products with coin prices

2. Enter delivery address

4. ✅ Coins deducted, order created

1. Go to "My Orders"
2. See redemption status

2. See stats, achievements, level
3. ✅ View progress
